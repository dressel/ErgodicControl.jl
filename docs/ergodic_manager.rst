=========================
Ergodic Manager
=========================

The ErgodicManager type keeps track of the spatial distribution.

Fields
=========
The ErgodicManager type has the following fields:
::
    K::Int
    bins::Int
    L::Float64
    cell_size::Float64
    hk::Matrix{Float64}
    phi::Matrix{Float64}
    phik::Matrix{Float64}
    Lambdak::Matrix{Float64}
    kpixl::Matrix{Float64}
